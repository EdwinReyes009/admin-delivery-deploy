# Admin Delivery Yummy Go

Main Dashboard to Yummy Go App, made with ReactJS
## Install Libraries

To install all libraries, run the following command

```bash
  npm install
```
```bash
  yarn install
```

## Running Test

To run project, run the following command

```bash
  npm run dev
```
```bash
  yarn dev
```
